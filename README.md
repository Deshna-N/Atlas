# 🌍 Atlas

Atlas is a full-stack travel companion that helps users plan adventures, track upcoming trips, save dream destinations, and visualize travel memories through a modern interactive interface.

---

## ✨ Features

### 🧳 Trip Management
- Create and manage trips
- Track destination, dates, budget, and notes
- Edit trip information
- Delete trips

### ⏳ Travel Countdown
- Automatically calculates trip status:
  - Adventure starts in X days
  - Adventure in progress
  - Adventure completed

### 🖼️ Destination Images
- Add custom destination images to trips
- Display images directly on travel cards

### 📊 Dashboard Analytics
- Total trips planned
- Total budget across all trips

### ⭐ Wishlist
- Save dream destinations
- Remove destinations from wishlist

### 🎨 Dynamic Themes
#### 🌙 Night Mode
- Animated fireflies
- Dark travel-inspired theme

#### ☀️ Day Mode
- Floating clouds
- Sky-to-sunrise gradient background

### 📱 Responsive Design
- Mobile-friendly layout
- Adaptive trip card grid

## 🏃 Running Locally

### Frontend

```bash

npm run dev
```

### Backend

```bash
python -m venv .venv
pip install -r requirements.txt
uvicorn main:app --reload
```
